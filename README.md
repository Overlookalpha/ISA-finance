# ISA Finance

## Objetivo

O ISA Finance é o sistema oficial de gestão patrimonial da IsaCred.

Ele foi desenvolvido para controlar automaticamente todo o fluxo financeiro da empresa, separando o patrimônio da empresa do patrimônio dos sócios conforme regras configuráveis.

## Regras de distribuição

### Faturamento até € 5.000

- Empresa: 76%
- Isaías: 12%
- Evelyn: 12%

### Faturamento acima de € 5.000

- Empresa: Percentual calculado automaticamente
- Evelyn: 12%
- Isaías: 20% sobre o valor excedente aos € 5.000

## Módulos

- Dashboard Financeiro
- Fluxo de Caixa
- Contas Bancárias
- Investidores
- Empréstimos
- Recebimentos
- Despesas
- Distribuição Automática
- Patrimônio da Empresa
- Patrimônio de Isaías
- Patrimônio de Evelyn
- Relatórios
- Auditoria Completa

## Objetivos do sistema

- Separar automaticamente os valores da empresa, Isaías e Evelyn.
- Registrar todas as movimentações financeiras.
- Controlar investidores e aportes.
- Gerenciar contas bancárias.
- Gerar relatórios financeiros.
- Manter histórico completo para auditoria.
